# Daily News

Live Demo: [https://daily-news-theta-nine.vercel.app/](https://daily-news-theta-nine.vercel.app/)

## Overview
Daily News is a real-time news application featuring AI-powered insights.

## Features
- **Live News Feed**: Automatically updates from top tech sources (BBC, NYT, Wired).
- **AI Chatbot**: Context-aware AI that can read articles and answer questions.
- **Keyless Architecture**: Uses RSS feeds and free AI endpoints for zero-config setup.
- **Premium Design**: Minimalist, monochrome UI with responsive interactions.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
